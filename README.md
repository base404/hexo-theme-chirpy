# Hexo Theme Chirpy Classic

> 简约、高颜值且功能丰富的 Hexo 博客主题，内置 **Hexo Theme Schema** 标准协议，支持 Hexo CMS 可视化配置、暗黑模式切换、Glassmorphism 磨砂玻璃预设、TOC 目录、代码高亮复制与阅读进度条。

---

## 🌟 核心特性

- 🎨 **4 种视觉预设 (Presets)**：
  - `chirpy` - Chirpy Classic (经典暗极灰 + 紫禁青)
  - `nordic` - Nordic Minimal (北欧极简极地蓝)
  - `cyberpunk` - Cyberpunk Neon (赛博霓虹暗黑风)
  - `glassmorphism` - Glassmorphism (磨砂玻璃质感与极光动感)
- 🎛️ **Hexo Theme Schema 标准驱动**：无缝兼容 Hexo CMS 可视化配置编辑器，表单拖拽与可视化渲染，修改自动持久化至 `_config.chirpy.yml`，零代码侵入 Hexo 物理源码。
- 📖 **交互体验**：
  - 文章页右侧悬浮 TOC 目录（随滚动实时高亮）
  - 页面顶部阅读进度条
  - 代码块一键复制按钮
  - 侧边栏模式切换 (Dark / Light)
  - Disqus / Giscus / Waline 评论区支持

---

## 🚀 安装使用

### 在 Hexo 项目中安装

```bash
cd themes
git clone https://github.com/base404/hexo-theme-chirpy.git chirpy
```

在 Hexo 根目录的 `_config.yml` 中修改：

```yaml
theme: chirpy
```

在 **Hexo CMS** 管理后台中：
1. 打开 `http://localhost:4001/#themes`
2. 点击 Chirpy 主题卡片上的【配置】按钮，即可唤起 Schema 动态配置表单！
